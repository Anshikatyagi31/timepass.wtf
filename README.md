# 🎉 Timepass.wtf

![Timepass.wtf Banner](/public/game-night-fun.png)

**[Live Demo →](https://timepass-wtf.vercel.app/)**  
An open-source gaming platform built with **Next.js**, designed to bring fun, challenges, and a perfect way to kill time—right in your browser!

---

## 🎮 About the Project

**Timepass.wtf** is a modern, casual gaming platform offering a variety of engaging browser-based games. Whether you’re into classic games like Tic-Tac-Toe or looking for brainy fun with Memory Games and Word Scrambles, there's something here for everyone.

---

## ✨ Features

- 🎲 **Game Variety**: Tic-Tac-Toe, Memory Game, Typing Test, Snake, Word Scramble, Crossword, and more
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- 🎨 **Beautiful UI**: Smooth animations and cohesive design using Tailwind CSS and Framer Motion
- 🧑‍🤝‍🧑 **Multiplayer Support**: Real-time multiplayer on selected games using Socket.IO
- 🌐 **Third-Party Integration**: Supports embedded games like Ludo from Gamezop
- ⚡ **Performance Optimized**: Fast loading and smooth gameplay experience

---

## 🛠️ Tech Stack

- **Frontend**: [Next.js](https://nextjs.org/), [TypeScript](https://www.typescriptlang.org/), [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Real-time**: [Socket.IO](https://socket.io/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/Anshikatyagi31/timepass.wtf.git

# 2. Install dependencies
npm install  # or yarn install

# 3. Run development server
npm run dev  # or yarn dev
